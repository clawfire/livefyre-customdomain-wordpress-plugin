Sync your users with your Livefyre Remote Profile Managment

== Description ==

With this plugin you can sync your Users with livefyre Remote profile managment daily (you can choose in the future) and provide an endpoint to livefyre which can retreive user specific data on demand.

We add functions in the future, see the planned functions and share your idea on [our uservoice](http://thibaultmilan.uservoice.com) and report bug / get help at the same URI .

== Installation ==

1. Upload the `livefyre-customdomain-wordpress-plugin` into your `wp-content/plugins/`
2. Go to your plugin manager and activate it
3. Go to the plugin config page to edit your settings details
